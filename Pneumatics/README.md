# Pneumatics Project

# 🔧 Synchronized Pneumatic Door & Crane System

## 📌 Project Overview

This project involved the design and implementation of a synchronized mechanical system consisting of two large welded steel doors and a rotating crane mechanism positioned between them. The system executes a fully automated repeating sequence controlled by an Arduino-based pneumatic control architecture.

### Automated Motion Sequence

1. Both steel doors open simultaneously (0° → 90°)  
2. Crane rotates outward approximately 45° between the doors  
3. Crane returns to starting position  
4. Both doors close simultaneously  
5. Cycle repeats

## 🏗 Mechanical Design & Fabrication

### Doors
- Fabricated from welded angle iron  
- Mounted to square tubing base structure  
- Actuated using two 5-inch stroke pneumatic cylinders  
- Designed for 0° to 90° rotational motion  
- Cylinders mounted directly behind doors for direct force transfer  

### Crane Mechanism
- Constructed from rebar  
- Mounted using a large pillow block bearing to allow smooth rotational motion  
- Actuated using one 5-inch stroke pneumatic cylinder  
- Designed to rotate approximately 45° outward and return  
- Rear-mounted pneumatic supported by reinforced steel box structure  

### Base Structure
- Fabricated from square steel tubing  
- Designed to resist torsional and dynamic loads  
- Provided rigid mounting for pneumatic components and hinges  

---

## 💨 Pneumatic System

- Three total 5-inch stroke pneumatic cylinders  
  - 2 for synchronized door actuation  
  - 1 for crane rotation  
- Pneumatic tubing connected to centralized air compressor  
- Electrically actuated solenoid valves  
- Designed for synchronized multi-actuator motion  

### Key Pneumatic Concepts Applied
- Stroke length selection  
- Force estimation  
- Flow rate considerations  
- Valve timing coordination  
- Pressure consistency across actuators

## ⚙️ Controls & Automation

The system was controlled using an Arduino-based control box.

Working alongside an electrical engineer, I:

The result was a fully automated, repeatable mechanical motion sequence integrating three actuators with coordinated timing.
